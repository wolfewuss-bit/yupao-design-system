## 四、设计规范网站开发

### 4.1 技术栈
- 纯静态 HTML + CSS + JS（无框架依赖）
- CSS Variables 实现亮色/暗夜主题切换
- localStorage 持久化主题选择
- 内联同步脚本防止暗夜模式闪白

### 4.2 页面结构（21 个页面）

**设计语言（带侧边栏）：**
- index.html — 介绍（含愿景、使命、核心价值观）
- design-principles.html — 设计原则
- design-values.html — 设计价值观
- colors.html — 颜色体系
- typography.html — 字体排版
- spacing.html — 间距系统
- radius.html — 圆角系统
- shadow.html — 阴影与层级
- iconography.html — 图标规范
- naming.html — 命名策略
- copywriting.html — 文案规范（整合产品文案规范 v1.5）
- accessibility.html — 无障碍规范
- motion.html — 动效规范
- layout.html — 布局原则
- responsive.html — 响应式适配
- dos-donts.html — Do's & Don'ts
- changelog.html — 变更日志

**独立页面（无侧边栏）：**
- components.html — 组件规范
- tokens.html — Design Tokens
- resources.html — 资源下载

### 4.3 功能特性
- 亮色/暗夜主题切换（右上角按钮，状态持久化）
- 左侧导航栏 + 当前页高亮 + SVG 图标
- 颜色色块点击复制色值
- 组件实时预览（按钮、输入框等用真实 CSS 渲染）
- 响应式布局，移动端自适应（隐藏侧边栏）
- 页面进入淡入动效 + 卡片 hover 上浮 + 表格行高亮
- 尊重系统「减少动态效果」设置

### 4.4 资源下载
- DESIGN.md 文件下载（JS fetch + blob 方案）
- 多端 Token 文件下载：
  - YPDesignTokens.swift（iOS）
  - YPDesignTokens.xml（Android）
  - YPDesignTokens.dart（Flutter）
- Figma 组件库外链
