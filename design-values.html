设计价值观

设计价值观是鱼泡直聘设计团队的核心信念，指导我们在每一个设计决策中做出正确的选择。

核心价值观

可信赖 Trustworthy

蓝色主色调传递专业、稳定的品牌形象。每一个设计细节都应让用户感到安心和信任。在招聘场景中，信任是最重要的基础。

高效率 Efficient

简洁直白的界面，减少认知负担，让用户快速完成核心操作。蓝领用户的时间宝贵，每一步操作都应尽可能简化。

亲和力 Approachable

圆润的视觉元素、温暖的品牌黄色点缀，降低使用门槛。面向蓝领群体的产品需要更加友好和易于理解。

包容性 Inclusive

设计面向所有用户，不因年龄、教育背景、身体条件而产生使用障碍。无障碍设计是基本要求，不是附加功能。

设计氛围

鱼泡直聘的设计氛围是所有视觉决策的基调：

维度描述
整体风格现代、简洁、功能导向的移动端 UI
信息密度中等偏高 — 列表卡片为主，信息层级清晰
视觉节奏大面积留白 + 紧凑的内容区域，呼吸感与效率并重
品牌调性蓝色科技感 + 黄色活力点缀，专业而不冰冷
暗夜模式深色背景降低视觉疲劳，色彩饱和度微调以适配深色环境

用户至上

鱼泡直聘服务于两大核心角色，设计需要同时满足双方需求：

求职者（找工作）

快速浏览职位信息、一键投递简历、实时沟通。界面需要简洁明了，操作路径尽可能短。

招聘者（找牛人）

高效筛选人才、管理职位、快速响应求职者。需要更丰富的信息展示和批量操作能力。

设计决策原则

  • 简单优于复杂：能用一步完成的操作，不要拆成两步
  • 清晰优于美观：信息传达的清晰度优先于视觉装饰
  • 一致优于创新：遵循已有的设计模式，除非有充分理由创新
  • 包容优于排他：设计应考虑边缘场景和特殊用户群体
  • 数据驱动：设计决策基于用户数据和反馈，而非个人偏好
