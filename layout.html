布局原则 Layout

鱼泡直聘的布局系统基于移动端优先的设计理念，采用灵活的栅格和间距体系构建一致的页面结构。

栅格系统 Grid System

属性值说明
设计稿宽度375px (1x)基于 iPhone 标准宽度
页面水平边距16px左右各 16px，内容区 343px
栅格列数灵活移动端不强制固定列数，按内容自适应
栅格间距12px多列布局时列间距
基础网格4px所有尺寸和间距基于 4px 网格

间距哲学 Spacing Philosophy

间距不仅是视觉美化，更是信息层级的表达工具：

  • 亲密性原则：相关元素间距小（8px），不相关元素间距大（24px）
  • 呼吸感：大面积留白 + 紧凑的内容区域，避免信息过载
  • 节奏感：重复使用相同的间距值，建立视觉节奏
  • 层级表达：间距大小反映信息层级关系
间距层级值含义
元素内部4–8px紧密关联的子元素（如图标与文字）
元素之间8–12px同一组内的元素（如列表项内部）
组之间16–20px不同功能组之间（如卡片之间）
区块之间24–32px不同内容区块之间（如模块分隔）

常见布局模式

1. 列表布局 List Layout

最常用的布局模式，用于职位列表、消息列表、设置页等。

属性值
列表项高度自适应，最小 52px
列表项内边距12px 16px
分割线1px，左侧缩进 16px
列表间距0（通过分割线分隔）

2. 卡片布局 Card Layout

用于职位卡片、人才卡片等信息聚合展示。

属性值
卡片间距12px
卡片内边距16px
卡片圆角12px
卡片阴影shadow/sm

3. 表单布局 Form Layout

用于登录、注册、信息编辑等表单页面。

属性值
表单项间距16px
标签与输入框间距8px
错误提示间距4px（输入框下方）
提交按钮间距24px（与最后一个表单项）

4. 详情页布局 Detail Layout

用于职位详情、个人主页等信息展示页面。

属性值
模块间距24px
模块标题与内容12px
底部操作栏高度56px + 安全区
底部操作栏内边距12px 16px

页面结构

鱼泡直聘标准页面由以下层级构成，从上到下依次为：

StatusBar 44px (iOS) / 24px (Android)
NavigationBar 44px · 返回 + 标题 + 操作
Content Area
自适应高度 · 可滚动
水平内边距 16px
内容宽度 343px
TabBar 49px · 图标 + 标签
SafeArea 34px (iPhone)
