鱼泡愿景

找工作，上鱼泡直聘

鱼泡使命

用科技和服务第一的理念，做招聘双方都满意的自动化招聘平台

鱼泡核心价值观

1. 企业增量

  • 多企业认证就能多职位（岗位），多好企业，就能多好职位（岗位）
  • 第一服务对象：企业，提供职位
  • 第二服务对象：求职者，提供简历

2. 招聘效果

  • 让招聘方准确快速的招聘到合适的人才，用户至上，服务第一，关注留存。

3. 合理价格

  • 合理价格是极致性价比的体现，用户复购的基础。
  • 利润是商业的本分，员工福利与公司再投资发展的基础。

4. 成果导向

  • 关注的是在实现目标的过程中个人的努力和最终呈现的结果，它隐含了两个概念：目标和个人的努力。
  • 这种导向强调的是在追求目标的过程中个人的成长和提升，以及最终成果的达成。
  • 成果导向的人更加注过程和结果的统一，他们认为通过努力和奋斗，最终会得到应有的回报和成果。

5. 简单高效

  • 因为简单，所以高效。
  • 因此必须拆分问题，化繁为简。包括人际关系也得简单，复杂了，就容易藏污纳垢，自然效率低。
  • 第一性原理就是第一性目标。

6. 求实求真

  • 求实：立足实际，尊重客观规律；
  • 求真：追求真理，探索本质。
  • 尊重是礼貌，和求实求真不违背，不是对立面。

设计氛围

维度描述
整体风格现代、简洁、功能导向的移动端 UI
信息密度中等偏高 — 列表卡片为主，信息层级清晰
视觉节奏大面积留白 + 紧凑的内容区域，呼吸感与效率并重
品牌调性蓝色科技感 + 黄色活力点缀，专业而不冰冷
暗夜模式深色背景降低视觉疲劳，色彩饱和度微调以适配深色环境
