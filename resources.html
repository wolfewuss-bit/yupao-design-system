DESIGN.md 文件说明

DESIGN.md 是鱼泡直聘设计系统的核心文档，以 Markdown 格式维护，具有以下特点：

特性说明
格式Markdown 纯文本，版本控制友好
内容完整的 Token 定义、组件规范、设计原则、无障碍要求
用途AI Agent 上下文、开发参考、设计交接、新人培训
维护随设计系统版本同步更新
位置项目根目录 /DESIGN.md
# DESIGN.md 使用方式 ## 1. AI Agent 上下文 将 DESIGN.md 作为 system prompt 或 context 提供给 AI Agent， 使其生成的代码自动遵循设计规范。 ## 2. 开发参考 开发者可直接查阅 DESIGN.md 获取 Token 值、组件规格等信息。 ## 3. 设计交接 设计师与开发者之间的交接文档，确保设计意图准确传达。 ## 4. 新人培训 新加入团队的成员可通过 DESIGN.md 快速了解设计系统。
