组件 Components

鱼泡直聘的核心 UI 组件规范，包含按钮、输入框、卡片、标签等常用组件的视觉与交互定义。

按钮 Button

按钮是最常用的交互组件，用于触发操作。鱼泡直聘定义了 5 种按钮类型。

按钮示例

状态变化
Default 默认正常显示
Pressed 按下opacity: 0.85，阴影降级
Disabled 禁用背景 #99D3FF，不可点击
Loading 加载显示 loading 动画，不可点击

输入框 Input

输入框用于收集用户文本输入，支持默认、聚焦、错误三种状态。

输入框示例

请输入正确的手机号
状态背景色边框色文字色
Default 默认#F5F7FC#E9EDF3rgba(0,0,0,0.85)
Focus 聚焦#F5F7FC#0092FFrgba(0,0,0,0.85)
Error 错误#F5F7FC#E8362Ergba(0,0,0,0.85)
Disabled 禁用#F5F6FA#E9EDF3rgba(0,0,0,0.25)

输入框规格：高度 44px，圆角 8px，内边距 12px，字号 14px Regular。
